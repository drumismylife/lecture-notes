현재시제
현재 직설법 — 지금 진행 중인 동작
능동태 현재 능동태 직설법 어미: -ω / -εις / -ει / -ομεν / -ετε / -ουσι(ν)
화자가 직접 동작을 행하며, 지금 진행 중이거나 반복되는 행위를 나타냅니다.
λύω (뤼오) → "나는 (지금) 풀고 있다" — 포로를 묶은 사슬을 지금 이 순간 풀고 있는 장면
인칭·수헬라어음가뜻어미
1인칭 단수λύ-ω뤼오나는 풀고 있다-ω
2인칭 단수λύ-εις뤼에이스너는 풀고 있다-εις
3인칭 단수λύ-ει뤼에이그/그녀는 풀고 있다-ει
1인칭 복수λύ-ομεν뤼오멘우리는 풀고 있다-ομεν
2인칭 복수λύ-ετε뤼에테너희는 풀고 있다-ετε
3인칭 복수λύ-ουσι(ν)뤼우씨(ν)그들은 풀고 있다-ουσι(ν)
예시 동사
γράφω (그라포) 나는 쓰고 있다
πιστεύω (피스튜오) 나는 믿고 있다
ἀκούω (아쿠오) 나는 듣고 있다
수동태 중간태 현재 수동/중간태 직설법 어미: -ομαι / -ῃ / -εται / -όμεθα / -εσθε / -ονται
수동태: 지금 다른 이에게 동작을 당하고 있음  |  중간태: 지금 자신을 위해 / 자신에게 영향이 돌아오는 동작
λύομαι (뤼오마이) → "나는 (지금) 풀리고 있다"  /  "나는 나를 위해 풀고 있다"  — 형태 동일, 문맥으로 구별
인칭·수헬라어음가뜻 (수동)뜻 (중간)
1단λύ-ομαι뤼오마이나는 풀리고 있다나는 나를 위해 풀고 있다
2단λύ-ῃ뤼에너는 풀리고 있다너는 너를 위해 풀고 있다
3단λύ-εται뤼에타이그는 풀리고 있다그는 자신을 위해 풀고 있다
1복λυ-όμεθα뤼오메싸우리는 풀리고 있다우리는 우리를 위해 풀고 있다
2복λύ-εσθε뤼에스떼너희는 풀리고 있다너희는 너희를 위해 풀고 있다
3복λύ-ονται뤼온타이그들은 풀리고 있다그들은 자신들을 위해 풀고 있다
예시 동사
πιστεύομαι (피스튜오마이) 나는 믿겨지고 있다
σῴζομαι (소조마이) 나는 구원받고 있다
미래시제
미래 직설법 — 앞으로 일어날 동작 (시제 표지: σ)
능동태 미래 능동태 직설법 표지: σ 삽입 → 어간+σ+능동어미
화자가 앞으로 직접 동작을 행할 것임을 나타냅니다.
λύσω (뤼소) → "나는 풀 것이다" — 미래에 묶인 것을 풀겠다는 의지 혹은 예측
인칭·수헬라어음가뜻어미
1단λύ-σ-ω뤼소나는 풀 것이다-σω
2단λύ-σ-εις뤼세이스너는 풀 것이다-σεις
3단λύ-σ-ει뤼세이그/그녀는 풀 것이다-σει
1복λύ-σ-ομεν뤼소멘우리는 풀 것이다-σομεν
2복λύ-σ-ετε뤼세테너희는 풀 것이다-σετε
3복λύ-σ-ουσι(ν)뤼수씨(ν)그들은 풀 것이다-σουσι(ν)
자음축약 예시

βαπτίζω → βαπτίσω (밥티소) 나는 침례를 줄 것이다
수동태 미래 수동태 직설법 표지: θησ (어간+θη+σ+수동어미)

λυθήσομαι (뤼데소마이) → "나는 풀릴 것이다" — 미래에 묶인 상태에서 해방될 것
인칭·수헬라어음가뜻
1단λυ-θή-σ-ομαι뤼데소마이나는 풀릴 것이다

3단λυ-θή-σ-εται뤼데세타이그는 풀릴 것이다
1복λυ-θη-σό-μεθα뤼데소메싸우리는 풀릴 것이다
2복λυ-θή-σ-εσθε뤼데세스떼너희는 풀릴 것이다

예시 동사

중간태 미래 중간태 직설법 표지: σ + 중간태어미 (미래수동과 구별됨!)
앞으로 자신을 위해 / 자신에게 이익이 돌아오도록 동작을 행할 것임을 나타냅니다.
λύσομαι (뤼소마이) → "나는 나를 위해 풀 것이다" — 미래 수동(λυθήσομαι)과 반드시 구별!

2단λύ-σ-ῃ뤼세너는 너를 위해 풀 것이다
3단λύ-σ-εται뤼세타이그는 자신을 위해 풀 것이다

2복λύ-σ-εσθε뤼세스떼너희는 너희를 위해 풀 것이다
3복λύ-σ-ονται뤼손타이그들은 자신들을 위해 풀 것이다
예시 동사

미완료
미완료 직설법 — 과거의 계속·반복된 동작 (가산모음 ε)
능동태 미완료 능동태 직설법 가산모음 ε- 접두 + 2차 능동어미
과거에 계속되거나 반복된 동작을 화자가 직접 행했음을 나타냅니다.

단순과거
단순(부정)과거 직설법 — 과거의 단순한 발생
제1단순과거 vs 제2단순과거: 번역은 둘 다 "~했다"로 동일합니다. 차이는 형태: 제1은 σα 표지(규칙변화), 제2는 어간 자체가 변화(불규칙, 어근에 따라 정해져 있음). 마치 영어의 walked(규칙)와 went(불규칙) 차이와 같습니다.
능동태 제1단순과거 능동태 직설법 가산모음 ε- + 어간 + σα + 2차어미
과거에 동작이 단순히 한 번 발생했음을 나타냅니다. 지속·반복의 의미 없음.
ἔλυσα (엘뤼사) → "나는 풀었다" — 특정 시점에 묶인 것을 한 번 풀었다는 사건
인칭·수헬라어음가뜻어미
1단ἔ-λυ-σα엘뤼사나는 풀었다-σα
2단ἔ-λυ-σας엘뤼사스너는 풀었다-σας
3단ἔ-λυ-σε(ν)엘뤼세(ν)그/그녀는 풀었다-σε(ν)
1복ἐ-λύ-σαμεν엘뤼사멘우리는 풀었다-σαμεν
2복ἐ-λύ-σατε엘뤼사테너희는 풀었다-σατε
3복ἔ-λυ-σαν엘뤼산그들은 풀었다-σαν
예시 동사
