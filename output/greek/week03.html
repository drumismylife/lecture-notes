01 현재 직설법(Present Indicative)이란?
정의
직설법(Indicative)은 지금 실제로 일어나는 사실을 진술하는 서법입니다.
"할 수도 있고 안 할 수도 있는 것"이 아니라 실제로 하는 행위를 나타냅니다.
해석: "나는 풀다" 또는 "나는 풀고 있다" — 두 가지 모두 가능.
"할 수도 있고 안 할 수도 있는 게 아니라 실제로 하는 거예요. 직설법이란 말은 지금 한다는 뜻이에요."
"동사 어미는 인칭 대명사를 포함한다 — 별도 주어 불필요."
02 어미 통합 정리 — 능동태 vs 수동태/중간태
⚡ 능동태 어미 (Active Endings)
1단 (나)-ω-오나
2단 (너)-εις-에이스너/당신
3단 (그)-ει-에이그/그녀
1복 (우리)-ομεν-오멘우리
2복 (너희)-ετε-에테너희
3복 (그들)-ουσι(ν)-우시(ν)그들
🔄 수동태/중간태 어미 (Passive/Middle)
1단 (나)-ομαι-오마이나
2단 (너)-ῃ-에너/당신
3단 (그)-εται-에타이그/그녀
1복 (우리)-ομεθα-오메타우리
2복 (너희)-εσθε-에스테너희
3복 (그들)-ονται-온타이그들
능동 1단 -ω(오) vs 수동 1단 -ομαι(오마이) — 수동이 더 길다!
능동 1복 -ομεν(오멘) vs 수동 1복 -ομεθα(오메타) — 수동이 더 길다!
가장 헷갈리는 2단: 능동 -εις(에이스) vs 수동 -ῃ(에) — 이 둘이 시험 핵심!
수동태 식별법: 이오타(ι)가 들어가거나 -ται / -εσθε / -ονται로 끝나면 수동태!
"이요타로 끝나거나 -ται, -θε, -νται 보이면 수동태."
"보며 외우는 거랑 쓰는 건 달라 — 반드시 종이에 써서 확인하세요."
03 중간태(Middle Voice) 정리
중간태란?
주어의 동작이 주어 자신에게 영향을 미칠 때 사용합니다 (셀프 액션).
예: λύομαι = "나는 나 자신을 풀다 / 나를 위해 풀다"
현재·미완료 시제에서는 수동태와 형태가 완전히 동일 → 문맥으로만 구별.
미래 시제에서는 수동태와 중간태 형태가 달라집니다.
"중간태는 지가 셀프로 영향을 미치는 거예요. 현재는 수동태와 중간태가 똑같아요."
"중간태는 시험에 안 낸다 — 현재·미완료에서 수동태와 형태 동일, 번역이 너무 힘들어서."
04 λύω (뤼오 — 풀다) 현재 완전 변화표
인칭/수형태음가해석
1인칭 단수λύω뤼오어간 λύ + ω나는 풀다 / 나는 풀고 있다
2인칭 단수λύεις뤼에이스어간 λύ + εις너는 풀다
3인칭 단수λύει뤼에이어간 λύ + ει그는 풀다
1인칭 복수λύομεν뤼오멘어간 λύ + ομεν우리는 풀다
2인칭 복수λύετε뤼에테어간 λύ + ετε너희는 풀다
3인칭 복수λύουσι(ν)뤼우시(ν)어간 λύ + ουσι그들은 풀다
인칭/수형태음가해석
1인칭 단수λύομαι뤼오마이어간 λύ + ομαι나는 풀리다
2인칭 단수λύῃ뤼에어간 λύ + ῃ너는 풀리다
3인칭 단수λύεται뤼에타이어간 λύ + εται그는 풀리다
1인칭 복수λυόμεθα뤼오메타어간 λύ + ομεθα우리는 풀리다
2인칭 복수λύεσθε뤼에스테어간 λύ + εσθε너희는 풀리다
3인칭 복수λύονται뤼온타이어간 λύ + ονται그들은 풀리다

문법현재 능동태 어미 6개
시험단수: -ω / -εις / -ει  (오 / 에이스 / 에이)
시험복수: -ομεν / -ετε / -ουσι(ν)  (오멘 / 에테 / 우시)
문법현재 수동태/중간태 어미 6개
시험단수: -ομαι / -ῃ / -εται  (오마이 / 에 / 에타이)
시험복수: -ομεθα / -εσθε / -ονται  (오메타 / 에스테 / 온타이)
시험시험 출제 포인트
시험2인칭 단수 구별 — 능동 -εις(에이스) vs 수동 -ῃ(에)
시험주어진 형태의 인칭·수·태 판별 (예: λύεται → 현재 수동태 3인칭 단수)
시험수동태 식별: 이오타(ι) 포함 또는 -ται / -εσθε / -ονται 어미
제외중간태 해석 — 현재 시제 한정으로 시험 출제 없음
