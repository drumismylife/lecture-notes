BLOCK 03 통일성과 다양성 · 어거스틴 명언
성경의 통일성과 다양성
정의
성경은 핵심 문제에서 한 목소리(통일성)를 내면서도, 독자·상황·강조점에 따른 다양한 관점(다양성)을 동시에 담음
예시
로마서(믿음 강조) vs 야고보서(행함 강조) — 모순이 아닌 강조점의 차이
마태복음(유대인) vs 마가복음(이방인) — 본질은 동일, 관점이 다름
주의
진보: "성경에 모순 있다"(다양성 과장) ↔ 보수: "다 똑같다"(다양성 무시) → 둘 다 경계
⭐ 시험 출제 예상 — 어거스틴 명언 (Unity · Liberty · Charity)
"본질적인 문제에서는 Unity(통일), 비본질적인 문제에서는 Liberty(자유), 모든 문제에서는 Charity(사랑)"

주의: 어거스틴 발언 여부는 학계 논란 있음 / 비본질의 "자유"는 상대방의 자유
Unity
통일성
본질적 문제
(구원·기독론 등)
Liberty
자유
비본질적 문제
상대방의 자유 인정
Charity
사랑
모든 문제
본질·비본질 상관없이
교수 설명
본질/비본질의 경계 기준이 사람마다 달라 적용이 쉽지 않음. 원칙은 모두 동의하지만 개별 사안(세례 vs 침례, 예배 시간, 악기 사용 등)에서 갈릴 수 있다. 본질에서는 목숨 걸고 하나가 되고, 비본질에서는 상대방에게 자유를 주어야 한다.
BLOCK 04 본문비평 (Textual Criticism)
본문비평 Textual Criticism
수천 개의 필사 사본 중 어떤 이문(異文)이 원문에 가장 가까운지를 재구성하는 학문적 작업
배경신약 원본 없음. 5,000개 이상의 필사 사본만 현존. 사본마다 이문(異文) 발생
이문(異文)사본마다 달라지는 단어·구·절·단락. 필사 실수·의도적 수정·첨가 등이 원인
경향후대로 올수록 사본이 두꺼워지는 경향 (첨가 경향 > 삭제 경향)
⭐ 시험 핵심 — 본문비평 3원칙 (암기: "오·어·짧")
원칙원문 가능성 높은 쪽이유
① 사본 연대오래된 사본필사 횟수 적을수록 오류 누적이 적음
② 표현 난이도어려운 이문필사자가 어려운 표현을 쉽게 바꾸는 경향
③ 분량짧은 이문보탤 때 죄의식이 적어 첨가 경향이 강함
KJV vs NIV·ESV
영어성경의 나이 ≠ 참고한 헬라어 사본의 나이. KJV(1611년)는 상대적으로 후대 사본 사용. 이후 더 오래된 헬라어 사본 발견 → NIV·ESV는 오래된 사본 기준 번역. 본문비평 관점에서는 NIV·ESV가 원문에 더 가까울 가능성. 단, KJV가 하나님 말씀으로 쓰임 받은 것은 부정하지 않음.
📌 마가복음 결말 문제 (막 16:9-20)
가장 오래된 사본: 16장 8절에서 끝남 → "여자들이 무서워 아무 말도 하지 못하더라"
후대 사본: 9-20절 추가 → 멋진 마무리

본문비평 3원칙 적용 → 8절 결말 = 오래된 + 어려운 + 짧은 → 원문일 가능성 높음
요한복음 7:53-8:11(간음한 여인)도 동일한 이유로 원문에 없었을 가능성.
※ 비본질적 문제 — 어느 쪽을 믿어도 무방
BLOCK 05 석의 vs 자기해석 · 역사비평
석의 vs 자기해석 Exegesis vs Eisegesis
석의 (釋義) ex-(밖으로) → 텍스트에서 의미를 끄집어내는 해석. 바람직한 방향
자기해석 eis-(안으로) → 자기 생각·경험·교리를 텍스트 안으로 집어넣는 해석. 경계해야 함
원칙 신학도 성경에서 나왔지만, 성경 위에 있을 수 없음. 신학이 성경 아래에 있어야 함
역사비평 (Historical Criticism)
본문의 역사적 배경(정치·사회·문화·종교)을 분석해 원래 의미를 파악하는 방법. 성경 비판이 아님. 예: 티베리우스 황제의 동전 글귀를 알아야 "가이사의 것은 가이사에게" 본문을 제대로 설교할 수 있음. 역사비평 = 석의를 돕는 도구.

💬 토론 주제 1 — 본문비평과 성경의 권위
1본문비평은 성경의 권위를 약화시키는가, 아니면 강화하는 도구인가?
논점: 원문에 더 가까이 가는 도구 vs 이문의 존재 자체가 "불완전성" 주장의 빌미 / 중요한 교리에는 이문이 없음

논점: 본문비평 원칙(오래된·짧은·어려운) vs 전통 텍스트 수용 / 비본질적 문제로 볼 수 있는가 / 설교에 어떤 영향을 미치는가

논점: 영어성경의 번역 연도 vs 참고 헬라어 사본의 나이 / 교인들의 KJV 선호를 존중하면서 학문적 설명을 어떻게 병행할 것인가
💬 토론 주제 2 — 통일성, 자유, 사랑 (어거스틴 명언 적용)

논점: 예배 음악, 세례/침례 방식, 예배 시간, 교단 교리 / 어떤 기준으로 본질을 구분할 것인가

논점: 종교다원주의로 흐를 위험 / 구원의 문제는 절대 비본질이 될 수 없음 / "자유"의 한계선은 어디인가
3나의 설교/목회에서 석의를 방해하는 자기해석의 요소는 무엇인가?
논점: 개인 경험, 교단 교리, 설교 습관 / 완전한 객관성이 가능한가 / 성령의 조명과 학문적 방법을 어떻게 통합할 것인가
